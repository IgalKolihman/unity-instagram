# unity_instagram

Unity Instagram website.

![profile example](assets/example.png)

## How to run

1. Clone this repository
2. `cd` into `build/web`

```bash
cd build/web
```

3. Run python http server

```bash
python -m SimpleHTTPServer 80 # for python2
```

```bash
python -m http.server 80 # for python3
```
